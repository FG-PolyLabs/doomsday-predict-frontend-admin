GCS Bucket
fg-polylabs-doomsday
Primary data store for exported JSON (index.json, events/). GCS is the authoritative fallback when jsDelivr CDN is stale.
Cloud Run Services
doomsday-api
REST API for CRUD operations on markets data. Validates Firebase ID tokens and restricts to allowed emails.
Cloud Run Jobs
doomsday-polymarket
Fetches YES/NO token price history from Polymarket for all active markets and writes deduplicated snapshots to BigQuery.
doomsday-exporter
Reads market_snapshots from BigQuery and publishes static JSON to GCS and the data repo. Run after polymarket or on demand.
Cloud Scheduler
doomsday-daily
Triggers doomsday-polymarket daily at 01:00 UTC with --all --yesterday --active-only. Runs as doomsday-runner SA.
BigQuery
fg-polylabs.doomsday
Dataset containing markets (config) and market_snapshots (price history, deduplicated on event_slug + snapshot_timestamp).
